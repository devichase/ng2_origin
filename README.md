* TypeScript
* Webpack
* Responsive layout
* High resolution
* Bootstrap 4 CSS Framework
* Sass
* Angular
* jQuery
* Charts (Chartist, Chart.js)
* Maps (Google, Leaflet, amMap)
* ...
